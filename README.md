## Features

- Browse random paintings from world-class museums
- Set any painting as your desktop wallpaper with one click
- Switch between art sources via the menu bar
- Minimal, distraction-free interface
- Cross-platform (macOS, Windows, Linux)

## Art Sources

- **Metropolitan Museum of Art** — Over 13,000 public domain paintings
- **Art Institute of Chicago** — Thousands of masterworks from their collection

## Usage

1. Open the app
2. Click the **refresh** button to browse random paintings
3. Click the **set wallpaper** button to apply the current painting
4. Use the **Source** menu to switch between museums

